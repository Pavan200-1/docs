When you download the Script from DroidOXY, you will get an archive file in .zip extension. You need to unzip that archive.  When done, you should see New upload folder (Folder name can be changed from version to version).

Please, follow the steps below to install the Script.

**Step 1** - Log into your hosting space (server) via Cpanel or FTP client.\
\
**Step 2** - In extracted archive folder, find **New upload** folder\
\
**Step 3** - Upload the **Contents of New upload folder** to your server .\
\
**Step 4** - After uploading the files you should go to

> [www.your-domain.com/install/](http://www.your-domain.com/install/)

to **begin installation** ( just add **/install/** to the end of your url ). Then you will be shown the below Screen. If All Server Requirements are Supported, Click Next to Proceed.

![Drodioxyimage Pn](/images/drodioxyimage.png)

**Step 5** - Enter your **Database Connection** details and Click **Next** ( database tables are created automatically ).

![Installation 2 Pn](/images/installation-2.png)

**Step 6** - Enter your Desired **Admin Login Credentials** and Click **Install Script** ( see screenshots below ).

![Installation 3 Pn](/images/installation-3.png)

**Step 7** - If Script is installed, then you'll be shown the below Screenshot.

![Installation Sucess Pn](/images/installation-sucess.png)

**Step 8** - Installation is Completed, but before using it, you need to **Remove/Rename** the install folder.

![Remove Install Pn](/images/remove-install.png)

**Congratulation, you are now ready to use Pocket GPT Script \!\!**
