## Getting Started

To Get Started with Rubicko offerwall, you’ll need a Rubicko Account

1. Rubicko Account
   - [Create Rubicko Account](https://rubicko.com/dashboard/signup.php)
2. Postback URL for Rubicko Network
   - Login to Admin Interface (Pocket Script instance)
   - Navigate to `Settings -> Postbacks S2S` on the admin sidebar
   - Copy the Rubicko Postback URL as shown in the admin interface

### Getting Rubicko Id's

To Intergrate the offerwall, you’ll need the following Id's

- Publisher ID
- Offerwall ID
- Secret ID

Please follow the below steps to get the required Id's

1. Login into your **Rubicko Dashboard**  and Copy the **_Publisher Id_**

   ![Rubicko 1 Pn](/images/Networks/Rubicko/rubicko_1.png)
2. Click on Offerwalls and **Create New** button.

   ![Rubicko 2 Pn](/images/Networks/Rubicko/rubicko_2.png)
3. Fill out the form and Click on the **Create Offerwall** button.

   ![Rubicko 3 Pn](/images/Networks/Rubicko/rubicko_3.png)
4. After Creating Offerwall, Copy the **_Offerwall Id_** and Click on Edit Offerwall

   ![Rubicko 4 Pn](/images/Networks/Rubicko/rubicko_4.png)
5. Scroll down to the Secret Id and Copy the Offerwall **Secret ID**

   ![Rubicko 5 Pn](/images/Networks/Rubicko/rubicko_5.png)

### Postback Test

Follow the below steps to make a postback test on the Rubicko platform

1. Go to `Dashboard -> Postback Test`
2. Scroll down to Postback Test Section
3. Input an Valid **User Id** to test the postback button
4. That's it your postback was sent successfully

![Rubicko 7 Pn](/images/Networks/Rubicko/rubicko_7.png)

## Integrating Offerwall

1. Login to your Dashboard
2. Navigate to `Offerwalls -> All Offerwalls`
3. Select and `Edit Rubicko Offerwall` and scroll down to Id’s Section
4. Paste the copied `Pub Id`, `Offerwall Id` and `Secret id` and click on the **Save Changes** button to complete the integration

   ![Rubicko 6 Pn](/images/Networks/Rubicko/rubicko_6.png)
