## **Getting Started** :

To Intergrate Adscend Media Offerwall you’ll need the following:

1. Adscend Media Account
   - Create Adscend Media Account

     https://adscendmedia.com/publishers/apply

## Getting Adscent Media ID's

To Intergrate Adscent Media offerwall, you’ll need the following Ids :

- Publisher ID
- Offerwall ID

Please follow the below steps to get the required Ids

1. Login to your **_Adscent Dashboard_** and Go to `Offerwall -> Create New Offerwall`

   ![Adsc1 Pn](/images/adsc1.png)
2. Go to Info & Settings tab, Enter all basic information and Save it

   ![Adsc2 Pn](/images/adsc2.png)
3. Copy **_pub Id_** and **_offerwall Id_** under **_INTEGRATION TAB_**  and paste it on your admin dashboard/web panel  `Settings -> offerwalls page`.

![Adsc3 Pn](/images/adsc3.png)

**That's it ..\!**
