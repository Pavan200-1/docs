## **Getting Started** :

To Intergrate Adscend Media Offerwall you’ll need the following:

Adscend Media Account

- Create Adscend Media Account

  https://adscendmedia.com/publishers/apply

Login to your **_Dashboard_**

![Adsc1 Pn](/images/adsc1.png)

Create a **_New Offerwall_**

![Adsc2 Pn](/images/adsc2.png)

Save it and then go to **_INTEGRATION TAB_**

![Adsc3 Pn](/images/adsc3.png)

Note the **_pub Id_** and **_offerwall Id_** and paste it on your admin dashboard/web panel

`Settings --> offerwalls page`.

**That's it ..\!**
