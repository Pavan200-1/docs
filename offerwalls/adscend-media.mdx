## Requirements :

To Intergrate, create an account with Adscend Media using the link below :

https://adscendmedia.com/publishers/apply

Login to your **dashboard**

![Adsc1 Pn](/images/adsc1.png)

Create a new offerwall

![Adsc2 Pn](/images/adsc2.png)

Save it and then go to **INTEGRATION TAB**

![Adsc3 Pn](/images/adsc3.png)

Note the **pub Id** and **offerwall Id** and paste it on your admin dashboard/web panel

**Settings --\> offerwalls page**.

**That's it ..\!**
