## Getting Started

To Intergrate Wannads offerwall, you’ll need the following:

1. Wannads Account
   - Create an Account

     https://publishers.wannads.com?referral_code=F0YRQR
2. Postback URL for Wannads Network
   - Login to Admin Interface
   - Navigate to `Admin Panel -> Settings -> Postbacks S2S`
   - Copy the Wannads Postback URL as shown in the admin interface
   - The Postback URL may look something similar to

     ```markdown
     http://yoururl.com//postbacks/wannads.php
     ```

After Successful Login into your **Wannads DashBoard**, you will find the below page, then go to **Create App**

![Wannads 1 Pn](/images/wannads_1.png)

Create a new App by giving name and url..

![Wannads 2 Pn](/images/wannads_2.png)
