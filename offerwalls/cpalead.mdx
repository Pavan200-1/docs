## Getting Started

To Intergrate Cpalead offerwall, you’ll need the following:

1. Create an cpalead account

   https://cpalead.com/get-started.php?ref=765337
2. Postback URL for Cpalead Network
   - Login to Admin Interface
   - Navigate to `Settings -> Postbacks S2S` on the admin sidebar
   - Copy the Cpalead Postback URL as shown in the admin interface
   - The Postback URL may look something similar to

     ```markdown
     http://yoururl.com/postbacks/cpalead.php?subid=&subid2=&virtual_currency=
     ```

Please follow the steps below to create an offerwall

1. Login to your **_Cpa Lead DashBoard_** and Go to `Offer Wall -> Create Offer Wall`

   ![Cpalead 1 Pn](/images/cpalead_1.png)
2. Create a **_New OfferWall_** by selecting a theme of your choice

   ![Cpalead 2 Pn](/images/cpalead_2.png)
3. Give your App Details ... like below and save it ..

   ![Cpalead 3 Pn](/images/cpalead_3.png)
4. Go to `Offer Wall -> Manage` and click on View Code button of your App.

   ![Cpalead 4 Pn](/images/cpalead_4.png)
5. Copy the Direct URL and paste it in your `Admin Panel -> Settings -> OfferWalls -> CpaLead`**.**

   ![Cpalead 5 Pn](/images/cpalead_5.png)

## Postback Test

Follow the below steps to make a postback test on the Cpalead platform

1. Go to `Postback -> Configuration` on the dashboard
2. Input an valid sub id from your application to test the postback
3. Paste your postback url in the field and the password is optional

   ![Cpalead 6 Pn](/images/cpalead_6.png)

**That's it.. \!**
